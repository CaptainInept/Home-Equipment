🛠 Maintenance Entry Standard

Use this format everywhere (append‑only):

YYYY-MM-DD – Action performed – Result / condition

✅ Quick Entry Checklist

  • Date in ISO format (YYYY-MM-DD)
  • One primary action word: Inspected / Tested / Cleaned / Flushed / Adjusted / Repaired / Replaced / Installed / Failed
  • Outcome in plain language (Normal / No issues / Monitor / Improved / Failed)
  • No sensitive info (no serials, invoices, account numbers, access codes)

✅ Examples

  • 2026-04-18 – Water heater flushed – Moderate sediment
  • 2025-11-02 – Pressure relief valve tested – Normal
  • 2024-01-10 – Anode rod replaced – Original rod 40% depleted
